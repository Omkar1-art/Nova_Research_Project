# Nova Research Project

AI-powered research platform built using React, Node.js, Express and MongoDB.

## Features

- AI Research Assistant
- Research Paper Analysis
- User Authentication
- File Upload Support
- Modern UI

## Project Structure

frontend/
backend/

## Installation

### Clone Repository

git clone https://github.com/Omkar1-art/Nova_Research_Project.git

### Frontend Setup

cd frontend
npm install
npm run dev

### Backend Setup

cd backend
npm install
npm start

## Environment Variables

Create a .env file inside backend folder.

Example:

OPENROUTER_API_KEY=your_api_key_here

## Technologies Used

- React.js
- Node.js
- Express.js
- MongoDB
- OpenRouter API
